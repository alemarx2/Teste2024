Estruturas HTML

Nas estruturas HTML por padrão, a largura é igual a 100%.
Para modificar este PADRÃO precisamos utilizar CSS.
Dentre os atributos CSS mais utilizados para mudar este padrão temos : margin, border,padding, position.
As novas TAG´s do html nos trazem novas semanticas que facilitam a leitura do código.
O CSS3 aqui também é indicado e melhora a leitura do código uma vez que limpa e otimiza o código.
